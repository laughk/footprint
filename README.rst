[WIP] FootPrint
===============

FootPritnt(足跡) is summary generator for Github activity.

inspire of `furik <https://github.com/pepabo/furik>`_ .

Install
----------

.. code:: console

    $ pip install git+https://github.com/laughk/footprint

Usage
--------

.. code:: console

    $ python  footprint.py [-h] [-f FROM_STR] [-t TO_STR]
